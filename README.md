# First project with Node.js

> #### Description

This project aims to learn how to create a server, add users, and implement other functionalities.

> #### Ideas

- [x] Create a server
- [x] Create a user
- [x] Save the users to a JSON file
- [ ] The user cannot register with a duplicate email
- [ ] Encrypt password
- [ ] Create tests
